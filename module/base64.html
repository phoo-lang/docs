use base64

b64.encode ( s → s )

Encode the string using base64.

b64.decode ( s → s )

Decode the string using base64. Incorrect padding (=) throws an error.

b16.encode ( s → s )

Encode the string using base16 (i.e. hexadecimal).

b16.decode ( s → s )

Decode the string using base16 (i.e. hexadecimal).
